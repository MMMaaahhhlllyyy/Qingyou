# Qingyou
## 轻游----带你游临港
### 项目介绍：<br>
轻游APP旨在通过打造一个临港地区的旅游平台，为外来游客提供个性化的游玩服务，为临港地区大学生提供兼职导游的就业机会。<br>
### 核心实现功能：<br>
临港旅游热门景点、美食、酒店推荐<br>
附近公园、健身房、游乐设施等休闲区域推荐<br>
旅游攻略分享<br>
